Strategic Inventory Analysis & Decision Support System (ABC-Pareto Automation)
This project is a VBA-based automation solution designed to optimize inventory management for businesses (e.g., hospitality, retail, or distribution centers). It eliminates manual data processing and provides data-driven insights into which products are critical for business profitability.

Project Overview
In most businesses, a small fraction of products (Category A) generates the majority of revenue. This tool applies the Pareto Principle (80/20 Rule) to:
Analyze large datasets in seconds.
Classify products into A, B, and C categories based on their cumulative impact.
Generate dynamic visual dashboards for management reporting.
Technical Features
Dynamic Data Engine: Automatically calculates total revenue by processing Quantity and Unit Price inputs.
Smart Classification: Implements an algorithm to calculate cumulative percentage and assign ABC categories dynamically.
Robust Error Handling: Designed with "Type Mismatch" protection to handle non-numeric data or empty cells without crashing.
Automated Visualization: Instant generation of inventory value charts with a single click.
User-Centric Design: Simple "Run Analysis" button interface, making it accessible for non-technical users.
Engineering Impact (KPIs)
Operational Efficiency: Reduced weekly manual analysis time by 95%.
Inventory Optimization: Enabled management to prioritize "Category A" items, significantly reducing stock-out risks.
Data Integrity: Eliminated human error associated with manual calculations and spreadsheet sorting.
How to Use
Open the abc_for_macro.xlsm file.
Enter your product names, quantities, and unit prices in the "Veri" (Data) sheet.
Click the "Run ABC Analysis" button.
Review the color-coded classification and the automatically generated chart on the right.
